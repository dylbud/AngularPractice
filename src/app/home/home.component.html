I am currently learning Angular and more front-end development in general. In this project, I am working through the app ideas listed here:

  • 15 App Ideas to Build and Level Up your Coding Skills

Listed and described below are the ones I've completed so far. These are also in the navigation menu above. :)

This project was deployed using Azure DevOps Pipelines.

Countdown Timer

Goal:

  • Enter event data in form to view a count down to that event.

What I learned:

  • Observables
  • RxJs Operators
    • Pipe
    • Map
    • Take
  • Ngx-bootstrap
    • Date picker
    • Time picker

Flip Image

Goal:

  • Be able to flip an image right/left/up/down

What I learned:

  • CSS
    • Transform
    • Transition
    • Perspective
  • The “Aha!” moment:
    • Realizing that rotating 360 degrees yields the same result as 0 degrees

Notes

What I learned:

  • Local Storage
  • Ngx-Bootstrap

  • Reactive Forms

Recipes

What I learned:

  • Transforming/mapping data

Quizzler
